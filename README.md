# Информационный сервис

Простая программа для получения информации по IP-адресу и курсам криптовалют.

---

## Что делает

-  По IP-адресу:  показывает страну, город, регион, координаты и провайдера
-  По криптовалюте:  показывает текущую цену в USD

---

## Как использовать

1. Введите IP-адрес (например,  8.8.8.8 ) или символ криптовалюты (например,  btc )
2. Нажмите кнопку  "Поиск" 
3. Результат появится в окне программы

---

## Какие API используются

| Что | API | URL |
|-----|-----|-----|
| Геолокация по IP | ip-api.com |  http://ip-api.com/json/{ip}  |
| Курсы криптовалют | CoinGecko |  https://api.coingecko.com/api/v3/simple/price?ids={coinId}&vs_currencies=usd  |

 API ключи не нужны!  Всё работает бесплатно.

---

## Что можно вводить

### Для проверки IP:
-  8.8.8.8  - Google DNS
-  1.1.1.1  - Cloudflare
-  188.186.255.154  - ваш IP

### Для проверки криптовалют:
-  btc  - Биткоин
-  eth  - Эфириум
-  sol  - Солана
-  doge  - Догикоин
